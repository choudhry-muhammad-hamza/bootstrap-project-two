# Bootstrap Project Two 🚀

This is a simple responsive website built using **HTML CSS and Bootstrap**.  
The project is created for practice and to improve frontend development skills.

---

## 📌 Features

- Fully responsive website using Bootstrap 5
- Modern Navbar with dropdown menu
- Interactive Image Carousel (Slider)
- Services section with icons (Bootstrap Icons)
- Skills/progress bars for development growth
- Contact form with input validation UI
- Embedded Google Maps location
- FAQ section using Bootstrap Accordion
- Clean and modern footer design
- Mobile-friendly layout (responsive grid system)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5

---

## 📂 Project Structure

index.html
style.css

---

## 🚀 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Explore different pages
